# 注册登录功能测试用例

## 文档说明

**版本**：v2.1（功能优化补充版）
**更新日期**：2026-06-16
**覆盖页面**：登录页 / 注册页 / 填写资料页 / 登录态分流。

---

## 1. 登录页

| 编号 | 场景 | 前置条件 | 操作 | 预期结果 |
|---|---|---|---|---|
| LOGIN-001 | 管理员登录成功 | 无登录态 | 输入 `admin` / `cc19980905` 点击登录 | Toast "管理员登录成功"，跳转管理端商品列表 |
| LOGIN-002 | 管理员密码错误 | 无登录态 | 输入 `admin` / 错误密码 | 密码字段提示"密码错误，请重新输入" |
| LOGIN-003 | 普通用户登录成功 | 已存在正常用户 | 输入正确账号密码 | Toast "登录成功"，跳转用户端 IP 分类页 |
| LOGIN-004 | 账号为空 | 无 | 密码有值，账号为空点击登录 | 账号字段提示"请输入账号" |
| LOGIN-005 | 密码为空 | 无 | 账号有值，密码为空点击登录 | 密码字段提示"请输入密码" |
| LOGIN-006 | 账号格式错误 | 无 | 输入非 admin 且不符合 6-12 位数字字母组合的账号 | 账号字段提示"账号需为6-12位数字和字母组合" |
| LOGIN-007 | 密码格式错误 | 无 | 输入含特殊符号或长度不符的密码 | 密码字段提示"密码需为6-12位数字或字母" |
| LOGIN-008 | 账号不存在 | 云数据库无该账号 | 输入未注册账号 | 账号字段提示"账号不存在，请先注册" |
| LOGIN-009 | 账号禁用 | 用户 `status=disabled` | 输入正确账号密码 | Toast "账号已被禁用，请联系管理员" |
| LOGIN-010 | 已登录自动分流 | 已有登录态 | 进入登录页 | 自动跳转对应角色首页 |
| LOGIN-011 | auth 云函数未部署 | 未部署 auth 云函数 | 点击登录 | Toast "云函数未部署，请先上传 auth 云函数" |
| LOGIN-012 | 登录网络异常 | 网络不可用或云函数超时 | 点击登录 | Toast "网络异常，请稍后重试" |

## 2. 注册页

| 编号 | 场景 | 前置条件 | 操作 | 预期结果 |
|---|---|---|---|---|
| REG-001 | 注册成功 | 账号未存在 | 输入合法账号、密码、确认密码 | 创建普通用户，Toast "注册成功"，跳转填写资料页 |
| REG-002 | 账号为空 | 无 | 账号为空点击注册 | 提示"请输入账号" |
| REG-003 | 账号格式错误 | 无 | 输入不含数字或不含字母的账号 | 提示"账号需为6-12位数字和字母组合" |
| REG-004 | 密码为空 | 无 | 密码为空点击注册 | 提示"请输入密码" |
| REG-005 | 密码格式错误 | 无 | 输入特殊符号或长度不符密码 | 提示"密码需为6-12位数字或字母" |
| REG-006 | 确认密码为空 | 无 | 确认密码为空点击注册 | 提示"请再次输入密码" |
| REG-007 | 两次密码不一致 | 无 | 密码与确认密码不同 | 提示"两次输入的密码不一致" |
| REG-008 | 账号已存在 | 云数据库已有账号 | 输入已存在账号注册 | 提示"账号已存在，请更换账号" |
| REG-009 | 返回登录 | 位于注册页 | 点击返回登录入口 | 返回登录页 |
| REG-010 | 注册云函数异常 | auth 云函数不可用 | 输入合法信息点击注册 | Toast 展示 authService 返回的用户提示，不创建本地登录态 |

## 3. 填写资料页

| 编号 | 场景 | 前置条件 | 操作 | 预期结果 |
|---|---|---|---|---|
| PROFILE-001 | 默认昵称 | 注册后进入资料页 | 不操作 | 昵称默认回填账号，头像显示账号首字 |
| PROFILE-002 | 保存成功 | 已登录 | 输入合法昵称，可选头像，点击完成 | 保存资料，Toast "资料保存成功"，跳转用户端 IP 分类页 |
| PROFILE-003 | 昵称为空 | 已登录 | 清空昵称点击完成 | 提示"请输入昵称" |
| PROFILE-004 | 昵称格式错误 | 已登录 | 输入特殊符号昵称 | 提示"昵称仅支持中文、英文或数字" |
| PROFILE-005 | 昵称超长 | 已登录 | 输入超过 12 字昵称 | 提示"昵称不能超过12个字" |
| PROFILE-006 | 头像超 5M | 已登录 | 选择大于 5M 图片 | 提示"头像图片不能超过5M" |
| PROFILE-007 | 头像选择失败 | 已登录 | 图片选择失败且非取消 | 提示"头像上传失败，请重新选择" |
| PROFILE-008 | 资料保存云函数异常 | auth 云函数不可用 | 点击完成 | Toast "保存失败，请重试"或云函数用户提示，停留当前页 |

## 4. 登录态与退出

| 编号 | 场景 | 前置条件 | 操作 | 预期结果 |
|---|---|---|---|---|
| SESSION-001 | 普通用户退出 | 普通用户已登录 | 在我的页点击退出登录并确认 | 清除登录态，回到登录页 |
| SESSION-002 | 管理员退出 | 管理员已登录 | 在设置页点击退出登录并确认 | 记录操作日志，清除登录态，回到登录页 |
| SESSION-003 | 未登录访问我的页 | 无登录态 | 打开用户端我的页 | 跳转登录页 |
| SESSION-004 | 非管理员访问设置 | 非管理员登录态 | 打开管理端设置页 | 跳转登录页 |

## 5. 修改密码与管理员重置

| 编号 | 场景 | 前置条件 | 操作 | 预期结果 |
|---|---|---|---|---|
| PWD-001 | 修改密码成功 | 普通用户已登录 | 输入正确原密码、新密码、确认新密码 | 调用 auth 云函数成功，清除登录态，Toast "密码修改成功"，跳转登录页 |
| PWD-002 | 原密码错误 | 普通用户已登录 | 输入错误原密码 | Toast 展示云函数返回错误，停留当前页 |
| PWD-003 | 新密码与原密码相同 | 普通用户已登录 | 新密码等于原密码 | Toast "新密码不能与原密码相同" |
| PWD-004 | 两次新密码不一致 | 普通用户已登录 | 新密码与确认新密码不同 | Toast / 就近提示"两次输入的新密码不一致" |
| PWD-005 | 管理员重置密码成功 | 管理员登录 | 在用户管理中确认重置密码 | 调用 `adminResetPassword`，目标用户密码重置为 `123456` |
| PWD-006 | 非管理员重置密码 | 非管理员登录态 | 调用重置密码 | 返回无权限提示"暂无操作权限，请联系管理员" |
