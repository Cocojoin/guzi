# 管理端需求文档 · 设置

## 文档说明

本文档仅覆盖管理端的**设置模块**，用于给设计、开发、测试单独使用。

**文档范围**：操作日志 / 数据备份 / 数据导出。

**通用基线**：本文默认继承 `管理端需求文档.md` 中的通用规则，包括通用列表规则、通用搜索与筛选规则、二次确认弹窗规则、反馈提示样式规则、通用异常处理、通用显示格式。

**版本**：v2.0（开发同步版）
**关联原型**：`谷圈星社-寄售管理首页.html`、`谷圈星社-寄售管理简洁版.html`
**标注约定**：`[待确认]` 表示规则未定 / 需要业务方再确认；`[默认]` 表示采用通用规则。

# 四、设置

设置用于记录平台管理相关的基础配置、操作记录和数据处理能力。

## 0. 当前开发同步说明

截至 2026-06-05，设置模块已开发 4 个页面：

| 页面 | 路由 | 已实现能力 |
|---|---|---|
| 设置首页 | `admin/pages/settings/settings` | 管理员信息、操作日志 / 数据备份 / 导出数据入口、版本、帮助反馈、退出登录 |
| 操作日志 | `admin/pages/settings/operation-log/operation-log` | 关键词、周期、类型、人员、结果筛选；日志保留周期设置；清理过期日志 |
| 数据备份 | `admin/pages/settings/data-backup/data-backup` | 最近备份信息、自动备份开关、频率选择、手动备份并打开文件 |
| 导出数据 | `admin/pages/settings/export-data/export-data` | 用户 / 商品 / 统计 / 日志多选导出，时间范围选择，生成 Excel 文件 |

## 1. 操作日志

### 1.1 🟦 显示内容

| 字段 | 显示 |
|---|---|
| 操作时间 | YYYY-MM-DD HH:mm:ss |
| 操作人 | 用户昵称 + 用户 ID |
| 操作类型 | 见 §1.3 操作类型枚举 |
| 操作对象 | 商品编号 / 用户 ID / 记录 ID 等 |
| 操作结果 | 成功 / 失败 |
| 备注 | 如失败原因 |
| 异常行为标记 | 异常行为显示红色"!"图标，便于排查 |

### 1.2 🟨 筛选与查询

- 支持关键词搜索，范围包含：操作标题、操作对象、操作人、备注。
- 支持按周期筛选：本月 / 近三月 / 全部。
- 支持按操作类型筛选：全部类型 + 当前日志中出现过的类型。
- 支持按操作人筛选：全部人员 + 当前日志中出现过的操作人。
- 支持按操作结果筛选：全部结果 / 仅成功 / 仅失败。
- 筛选条件即时生效，多个条件取交集。

### 1.3 操作类型枚举（最小集合）

| 类型 | 说明 |
|---|---|
| 上传商品 | 单个 / 批量 |
| 编辑商品 | 字段变更明细 |
| 删除商品 | 单个 / 批量 |
| 修改商品状态 | 流转前状态 → 流转后状态 |
| 上架并填链接 | 单个 / 批量 |
| 提交结算 | 含商品清单 |
| 重置密码 | 目标用户 |
| 修改抽成 | 原值 → 新值 |
| 修改寄售权限 | 开 / 关 |
| 编辑用户信息 | 字段变更明细 |
| 导出数据 | 范围 / 范围说明 |
| 数据备份 | 自动 / 手动 |
| 登录 / 登出 | 异常登录标记为异常行为 |

### 1.4 🟧 提示规则

| 场景 | 文案 |
|---|---|
| 列表为空 | 暂无操作记录 |
| 加载失败 | 数据加载失败，请刷新后重试 |
| 修改保留周期成功 | 已改为[保留周期] |
| 清理过期日志成功 | 已清理 N 条 / 暂无过期日志 |

### 1.5 用途

- 排查问题、定位异常操作、辅助追溯数据变化。

### 1.6 日志保留规则

| 项 | 规则 |
|---|---|
| 默认保留周期 | 90 天 |
| 可选保留周期 | 7 天 / 30 天 / 90 天 / 180 天 / 永久保留 |
| 清理方式 | 进入日志页时按当前保留周期自动清理；管理员也可点击"清理过期日志"手动执行 |
| 存储口径 | 优先写入云数据库 `admin_operation_logs`，同时保留本地缓存兜底 |
| 本地缓存上限 | 最近 300 条 |

## 2. 数据备份

### 2.1 🟦 显示内容

| 元素 | 显示 |
|---|---|
| 最近一次备份时间 | YYYY-MM-DD HH:mm |
| 备份状态 | 成功 / 失败 / 进行中 |
| 备份结果 | 文件大小 / 数据条数（简要） |
| 自动备份开关 | 默认开启，可关闭 |
| 备份频率 | 每日 03:00 / 每周一 03:00 / 每月 1 日 03:00 |
| 手动备份按钮 | 主按钮 |

### 2.2 🟨 交互规则

| 操作 | 效果 |
|---|---|
| 点击手动备份 | Dialog 确认 → 按钮加载态 → 成功后刷新"最近一次备份"区块 |
| 切换自动备份开关 | 直接生效，Toast 提示当前状态 |
| 选择备份频率 | ActionSheet 选择频率，选择后立即保存并写入操作日志 |
| 手动备份成功 | 生成备份文件，刷新最近一次备份信息，并尝试打开文件 |
| 备份历史 | 当前版本不提供历史备份列表 |

### 2.3 🟧 提示规则

| 场景 | 文案 | 形式 |
|---|---|---|
| 手动备份前 | 确认立即执行数据备份吗？ | Dialog |
| 备份中 | 备份中…（按钮显示加载态） | 按钮 |
| 备份成功 | 备份完成 | Toast |
| 备份失败 | 备份失败，请重试 | Toast |

## 3. 导出数据

### 3.1 🟦 显示内容

| 元素 | 显示 |
|---|---|
| 导出范围 | 多选：用户数据 / 商品数据 / 统计数据 / 操作日志；默认勾选统计数据 |
| 时间范围 | 本月 / 近三月 / 自定义 / 全部；自定义需填写开始和结束日期 |
| 导出格式 | Excel（.xls） |
| 导出按钮 | 主按钮 |

### 3.2 🟨 交互规则

| 操作 | 效果 |
|---|---|
| 点击导出 | Dialog 二次确认 |
| 确认后 | 后台生成文件，进度条 / 加载态显示 |
| 完成 | Toast "文件生成成功"，并尝试打开导出文件 |
| 大文件场景 | 当前版本同步生成，不做异步任务与站内通知 |

### 3.3 🟧 提示规则

| 场景 | 文案 | 形式 |
|---|---|---|
| 未选范围 | 请至少选择一项导出范围 | Toast |
| 自定义时间不完整 | 请选择完整的自定义时间 | Toast |
| 自定义开始时间晚于结束时间 | 开始时间不能晚于结束时间 | Toast |
| 导出前 | 导出数据可能包含敏感信息，请确认是否导出？ | Dialog |
| 导出中 | 导出中…（按钮显示加载态） | 按钮 |
| 导出成功 | 文件生成成功 | Toast |
| 导出失败 | 导出失败，请重试 | Toast |
