# 四、设置

设置用于记录平台管理相关的基础配置、操作记录和数据处理能力。

## 0. 当前需求同步说明

截至 2026-06-12，设置模块需求包含 5 个页面：

| 页面 | 路由 | 目标能力 |
|---|---|---|
| 设置首页 | `admin/pages/settings/settings` | 管理员信息、IP管理 / 操作日志 / 数据备份 / 导出数据入口、版本、帮助反馈、退出登录 |
| IP管理 | `admin/pages/settings/ip-groups/ip-groups` | 新增IP、查看IP、分配商品、移除商品、删除IP |
| 操作日志 | `admin/pages/settings/operation-log/operation-log` | 关键词、周期、类型、人员、结果筛选；日志保留周期设置；清理过期日志 |
| 数据备份 | `admin/pages/settings/data-backup/data-backup` | 最近备份信息、自动备份开关、频率选择、手动备份并打开文件 |
| 导出数据 | `admin/pages/settings/export-data/export-data` | 用户 / 商品 / 统计 / 日志多选导出，时间范围选择，生成 Excel 文件 |

## 1. IP管理

### 1.1 页面目标
- 统一管理用户端“谷子集市”的 IP 分类。
- 支持平台手动新增 IP。
- 支持将商品分配到指定 IP、从 IP 中移除商品。
- 支持删除 IP。

### 1.2 设置首页入口
- 设置首页增加“IP管理”入口。
- 入口信息建议：
  - 标题：IP管理
  - 副标题：新增IP、调整商品归属、删除IP

### 1.3 数据口径与自动同步规则
- IP 管理以商品字段 `ip` 为主口径。
- 管理端每新增商品时，系统根据商品的 `ip` 字段自动同步对应 IP：
  - 若该 IP 不存在，则自动创建 IP。
  - 若该 IP 已存在，则该商品自动加入已有 IP。
- 管理端编辑商品时：
  - 若商品 IP 被修改，则商品应从旧 IP 迁移到新 IP。
  - 若新 IP 不存在，则自动创建。
- 批量新增商品时，同样适用上述自动同步规则。
- IP 名称按完整文本匹配，默认去除首尾空格后比较。

### 1.4 列表展示规则
- IP 管理列表每项至少展示：
  - IP 名称
  - 商品总数
  - 当前在售商品数
  - 当前角色数
  - 最近更新时间（可选）
- 默认排序建议：
  1. 当前在售商品数倒序
  2. 最近更新时间倒序
  3. IP 名称正序

### 1.5 新增 IP
- 管理员可手动新增 IP。
- 字段：
  - IP 名称（必填）
- 校验规则：
  - 不能为空
  - 去除首尾空格后存储
  - 不允许与现有 IP 重名
- 新增成功后立即刷新列表。
- 新增空 IP 后，即使没有商品，也在管理端列表显示。
- 空 IP 不影响用户端；用户端只展示有上架商品的 IP。

### 1.6 分配商品到 IP
- 在某个 IP 下支持“添加商品 / 分配商品”操作。
- 可选择 1 个或多个商品加入该 IP。
- 被分配商品的 `ip` 字段直接更新为目标 IP 名称。
- 若商品原本属于其他 IP，则视为迁移，不保留在原 IP 中。
- 分配成功后：
  - 当前 IP 商品数实时更新
  - 原 IP 商品数同步更新

### 1.7 从 IP 中移除商品
- 在某个 IP 下支持“移除商品”操作。
- 可选择 1 个或多个当前属于该 IP 的商品移除。
- 移除后，商品的 `ip` 字段清空。
- 被移除商品不会自动删除。
- 若商品已上架但被移除 IP，则该商品不再出现在用户端任何 IP 分类和对应列表中。

### 1.8 删除 IP
- IP 支持删除。
- 删除规则：
  - 仅允许删除“IP 内无商品”的空 IP
  - 只要该 IP 下仍存在商品，就不允许删除
  - 删除成功后，该 IP 不再出现在管理端和用户端
- 删除操作必须二次确认。
- 当 IP 下仍有商品时，删除按钮置灰或点击后直接提示不可删除原因。

### 1.9 用户端联动规则
- 用户端谷子集市首页仅展示“有上架商品”的 IP。
- 用户端 Banner 仅展示最近上新的上架 IP。
- 用户端点击某个 IP 后，进入该 IP 的商品列表页。
- 若某个 IP 被删除或其下商品全部不再上架，则该 IP 不再出现在用户端。

### 1.10 操作日志
- 以下操作需写入操作日志：
  - 新增 IP
  - 删除 IP
  - 分配商品到 IP
  - 从 IP 中移除商品
  - 商品编辑导致 IP 迁移
  - 商品新增自动创建 IP（可简化记入“新增商品”备注）

### 1.11 提示规则

| 场景 | 文案 |
|---|---|
| 新增 IP 成功 | IP 已新增 |
| IP 名称为空 | 请输入 IP 名称 |
| IP 重名 | IP 已存在，请勿重复添加 |
| 分配商品成功 | 已加入该 IP |
| 移除商品成功 | 已移出该 IP |
| 删除 IP 成功 | IP 已删除 |
| 删除受限 | 当前 IP 下仍有商品，无法删除 |
| 删除确认 | 确认删除该空 IP 吗？ |

## 2. 操作日志

### 2.1 显示内容

| 字段 | 显示 |
|---|---|
| 操作时间 | YYYY-MM-DD HH:mm:ss |
| 操作人 | 用户昵称 + 用户 ID |
| 操作类型 | 见 §2.3 操作类型枚举 |
| 操作对象 | 商品编号 / 用户 ID / 记录 ID 等 |
| 操作结果 | 成功 / 失败 |
| 备注 | 如失败原因 |
| 异常行为标记 | 异常行为显示红色"!"图标，便于排查 |

### 2.2 筛选与查询
- 支持关键词搜索，范围包含：操作标题、操作对象、操作人、备注。
- 支持按周期筛选：本月 / 近三月 / 全部。
- 支持按操作类型筛选：全部类型 + 当前日志中出现过的类型。
- 支持按操作人筛选：全部人员 + 当前日志中出现过的操作人。
- 支持按操作结果筛选：全部结果 / 仅成功 / 仅失败。
- 筛选条件即时生效，多个条件取交集。

### 2.3 操作类型枚举（最小集合）

| 类型 | 说明 |
|---|---|
| 上传商品 | 单个 / 批量 |
| 编辑商品 | 字段变更明细 |
| 删除商品 | 单个 / 批量 |
| 修改商品状态 | 流转前状态 → 流转后状态 |
| 上架并填链接 | 单个 / 批量 |
| 提交结算 | 含商品清单 |
| IP管理 | 新增IP / 删除IP / 分配商品 / 移除商品 |
| 重置密码 | 目标用户 |
| 修改抽成 | 原值 → 新值 |
| 修改寄售权限 | 开 / 关 |
| 编辑用户信息 | 字段变更明细 |
| 导出数据 | 范围 / 范围说明 |
| 数据备份 | 自动 / 手动 |
| 登录 / 登出 | 异常登录标记为异常行为 |

### 2.4 提示规则

| 场景 | 文案 |
|---|---|
| 列表为空 | 暂无操作记录 |
| 加载失败 | 数据加载失败，请刷新后重试 |
| 修改保留周期成功 | 已改为[保留周期] |
| 清理过期日志成功 | 已清理 N 条 / 暂无过期日志 |

### 2.5 日志保留规则

| 项 | 规则 |
|---|---|
| 默认保留周期 | 90 天 |
| 可选保留周期 | 7 天 / 30 天 / 90 天 / 180 天 / 永久保留 |
| 清理方式 | 进入日志页时按当前保留周期自动清理；管理员也可点击"清理过期日志"手动执行 |
| 存储口径 | 优先写入云数据库 `admin_operation_logs`，同时保留本地缓存兜底 |
| 本地缓存上限 | 最近 300 条 |

## 3. 数据备份

### 3.1 显示内容

| 元素 | 显示 |
|---|---|
| 最近一次备份时间 | YYYY-MM-DD HH:mm |
| 备份状态 | 成功 / 失败 / 进行中 |
| 备份结果 | 文件大小 / 数据条数（简要） |
| 自动备份开关 | 默认开启，可关闭 |
| 备份频率 | 每日 03:00 / 每周一 03:00 / 每月 1 日 03:00 |
| 手动备份按钮 | 主按钮 |

### 3.2 交互规则

| 操作 | 效果 |
|---|---|
| 点击手动备份 | Dialog 确认 → 按钮加载态 → 成功后刷新“最近一次备份”区块 |
| 切换自动备份开关 | 直接生效，Toast 提示当前状态 |
| 选择备份频率 | ActionSheet 选择频率，选择后立即保存并写入操作日志 |
| 手动备份成功 | 生成备份文件，刷新最近一次备份信息，并尝试打开文件 |
| 备份历史 | 当前版本不提供历史备份列表 |

### 3.3 提示规则

| 场景 | 文案 | 形式 |
|---|---|---|
| 手动备份前 | 确认立即执行数据备份吗？ | Dialog |
| 备份中 | 备份中…（按钮显示加载态） | 按钮 |
| 备份成功 | 备份完成 | Toast |
| 备份失败 | 备份失败，请重试 | Toast |

## 4. 导出数据

### 4.1 显示内容

| 元素 | 显示 |
|---|---|
| 导出范围 | 多选：用户数据 / 商品数据 / 统计数据 / 操作日志；默认勾选统计数据 |
| 时间范围 | 本月 / 近三月 / 自定义 / 全部；自定义需填写开始和结束日期 |
| 导出格式 | Excel（.xls） |
| 导出按钮 | 主按钮 |

### 4.2 交互规则

| 操作 | 效果 |
|---|---|
| 点击导出 | Dialog 二次确认 |
| 确认后 | 后台生成文件，进度条 / 加载态显示 |
| 完成 | Toast “文件生成成功”，并尝试打开导出文件 |
| 大文件场景 | 当前版本同步生成，不做异步任务与站内通知 |

### 4.3 提示规则

| 场景 | 文案 | 形式 |
|---|---|---|
| 未选范围 | 请至少选择一项导出范围 | Toast |
| 自定义时间不完整 | 请选择完整的自定义时间 | Toast |
| 自定义开始时间晚于结束时间 | 开始时间不能晚于结束时间 | Toast |
| 导出前 | 导出数据可能包含敏感信息，请确认是否导出？ | Dialog |
| 导出中 | 导出中…（按钮显示加载态） | 按钮 |
| 导出成功 | 文件生成成功 | Toast |
| 导出失败 | 导出失败，请重试 | Toast |
